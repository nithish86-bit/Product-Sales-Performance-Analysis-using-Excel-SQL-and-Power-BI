## 🚀 Key Highlights & Skills Demonstrated

### 📊 1. Spreadsheet Engineering (MS Excel)
* **Missing Data Imputation:** Utilized lookup functions to repair structural gaps across data tables.
* **Sales Contribution Metrics:** Built mathematical logic to calculate granular product performance metrics out of total gross revenue.
* **Aggregated Performance Reporting:** Calculated key business metrics such as **Average Order Value (AOV)** and isolated high-volume transactional drivers.

### 🗄️ 2. Structural Querying (SQL)
* Aggregated retail revenue across category fields using `SUM()` and `GROUP BY`.
* Calculated margin volatility using mathematical absolute variations (`ABS`).
* Used window functions (`DENSE_RANK()`) to filter specific performance percentiles (3rd and 4th top-selling categories).

### 📉 3. Interactive Business Intelligence (Power BI)
* Engineered a dynamic reporting engine visualizing Total Transactions, Discount Impacts, and Categorical Sales Proportions.
* Implemented cross-filtering slicers allowing real-time deep-dives by product type and discount bracket.
